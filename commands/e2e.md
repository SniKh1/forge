# E2E Command

Generate and run end-to-end tests with Playwright.

## What This Command Does

1. Generate Test Journeys
2. Run E2E Tests across browsers
3. Capture Screenshots/Videos on failures
4. Generate HTML reports

## When to Use

- Testing critical user journeys
- Verifying multi-step flows
- Testing UI interactions
- Validating frontend-backend integration

## Quick Commands

```bash
npx playwright test
npx playwright test --headed
npx playwright test --debug
npx playwright show-report
```

## Related Agent

Invokes `e2e-runner` agent at `~/.claude/agents/e2e-runner.md`
