# Defuddle

Use this when the user provides a URL and the goal is to ingest or analyze the page content cleanly, especially for note systems and vault workflows.

## Guidance

- Prefer markdown output
- Use it instead of noisier page fetches when a clean article/doc body is what matters
- Save cleaned content into notes or downstream workflows when useful
