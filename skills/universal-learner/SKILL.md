# Universal Learner

Use this when the task is to extract reusable prompt elements or evolve a prompt library rather than answer with a single prompt.

## Good Fit

- Learn patterns from one or many prompts
- Build reusable prompt libraries
- Group prompt components by domain
- Capture recurring structures for later generation
