# JSON Canvas

Use this when the task involves `.canvas` files, node-edge maps, or visual note structures.

## Focus Areas

- Canvas node and edge structure
- Unique IDs
- Visual layout and spacing
- Validation of node references and JSON integrity

## Guidance

- Keep IDs unique across nodes and edges
- Validate all edge references
- Use readable spatial layout instead of overlapping nodes
