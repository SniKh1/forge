# Obsidian Bases

Use this when the task is about `.base` files, filtered note views, formulas, or lightweight database workflows inside Obsidian.

## Focus Areas

- YAML-based `.base` files
- Filters and grouped views
- Table/cards/list/map style outputs
- Formula-backed computed fields

## Guidance

- Keep the file valid YAML
- Prefer clear filters and explicit property ordering
- Validate formula names and referenced properties
