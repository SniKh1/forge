# Prompt Analyzer

Use this when a prompt already exists and the task is to inspect or compare it rather than generate a new one.

## Good Fit

- Analyze the structure of an existing prompt
- Compare two prompt variants
- Recommend similar prompts
- Inspect prompt element usage and library statistics

## Guidance

- Favor analysis over generation
- Explain why prompts differ, not just that they differ
- Surface reusable structures and recurring elements
