# Intelligent Prompt Generator

Use this when the user wants a new prompt from scratch and needs structured generation rather than prompt review.

## Good Fit

- Generate prompts from a short goal
- Expand sparse prompt requests into richer prompt structure
- Choose prompt composition based on domain and visual/task intent

## Guidance

- Separate semantic intent from rendering/style details
- Fill in missing but necessary structure without drifting from user intent
- Prefer consistency and concrete visual/task attributes over vague adjectives
